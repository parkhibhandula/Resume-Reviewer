A smart AI-powered Resume Reviewer built using React, Node.js, and AI APIs.

It analyzes resumes against job descriptions, calculates a match score, and highlights missing skills, experiences, and improvements needed to boost your chances of getting shortlisted.

Features

Resume Parsing → Upload your resume and extract structured information.

Job Description Analysis → Paste a JD to compare requirements.

AI-Powered Match Score → Uses an AI model to generate a similarity score.

Missing Skills Suggestion → Identifies skills that are missing from your resume.

Actionable Feedback → Gives personalized tips to improve your resume.

Interactive UI → Built with React for a smooth and modern user experience.

Fast & Scalable Backend → Handles file uploads, API calls, and AI integration.
